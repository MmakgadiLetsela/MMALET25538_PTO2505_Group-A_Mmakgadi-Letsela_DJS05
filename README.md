# React Podcast App with Dynamic Routing and Navigation


## Project Overview
This project is a React-based podcast browsing application that allows users to explore podcasts using routing. 
It introduces routing using BrowserRouter in React. Users are able to click on podcast previews and be directed to a seperate detailed page.

## Technologies Used
- HTML
- CSS
- Javascript
- React
